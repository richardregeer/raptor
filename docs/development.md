# Development
## Requirements
 - Docker 
 - Make
 
## Environment setup
- Install [Docker](https://docs.docker.com/engine/install/)
- Install Make
```
# Unbutu:
sudo apt install build-essential
```
- Install Raptor development 
```
make install
```
Some available commands:
```
# Show all available commands
make help

# Run all tests
make test

# Lint code
make lint
```
